About This Prototype

This HTML/CSS prototype demonstrates the user interface of ZelTech Logistics' three core systems: Fleet Management System (FMS), Warehouse Management System (WMS), and Delivery Tracking System (DTS). The UI elements match the BPMNs and the unified database shown in the Enterprise Architecture Plan.
